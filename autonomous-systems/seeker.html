Seeker

An autonomous robot capable of following an object within its detection range. Low-layer programming on a STM32 microcontroller.

As demonstrated in the video bellow, this robot can follow an object located between 10cm and 40cm away from the ultrasound sensors. The speed of the robot varies depending on the distance from the object. It's three alternating sensors offer a 120Â° detection span as well as an approximate position. The program is very optimized, as it is only Low Layer code.

Key elements

C

The whole software was programmed in basic C. This language allowed low level interraction with memory.

Low Layer Programming

The code only uses low-level functions : read a registry, write a registry, compare two variables, start an interruption. There isn't any Hardware Abstraction Layer (HAL).

Microcontroller

This project was developped for a STM32-Nucleo-L053R8 microcontroller.
