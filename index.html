My projects

OS Layer in Python

A Python software simulating a windows manager capable of handling multiple programs running simultaneously.

Seeker

An autonomous robot capable of following an object within its detection range. Low-layer programming on a STM32 microcontroller.

CVE Checker

A Python software capable of detecting Common Vulnerabilities and Exposures amongst all the applications of a computer.

Hangman

A web application of the famous hangman game, in which you must guess a word. The whole app is deployable with Docker.

OT Demonstrator

A physical model train simulation demonstrating the consequences of a cybersecurity attack on an industrial network.

Picross

Picross game fully developped in basic C. This classic puzzle challenges players to reveal a hidden picture by filling in cells based on numerical clues.

Garage Structures

Program crafted in C designed to illustrate various data structures. Such as stacks, queues, and trees through interactive visualizations.

SW Advanced Algorithms

Explore various algorithms programmed in C such as Dijkstra's OSPF, A*, Bellman-Ford & Ford-Fulkerson with a Star Wars theme.

ESIEAbot Catapult

4-day crafted robot, equipped with a mini catapult for launching ping pong balls. This robot showcases basic engineering and programming.

CEM - Certificat d'Étude Musicales

The Certificat d'Études Musicales (CEM) is a french diploma that recognizes comprehensive musical studies, awarded by conservatories after twelve years of training in theory, instrumental, vocal practice, and ensemble performance. It validates a high level of musical proficiency and understanding.
