CVE Checker

A Python software capable of detecting Common Vulnerabilities and Exposures amongst all the applications of a computer.

Applications with outdated versions have security vulnerabilities. Therefore, the majority of computers are vulnerable to numerous cyberattacks. Our software, designed for those overlooked in cybersecurity, offers a simple and accessible solution. It can detect outdated applications on a Windows system that have listed security vulnerabilities (CVE).
